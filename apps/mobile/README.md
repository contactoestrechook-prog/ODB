# apps/mobile — App de cliente (Expo / React Native)

Scaffold previsto: `npx create-expo-app mobile`.

Features: registro con verificación biométrica (SDK Didit: DNI + selfie contra RENAPER, valida mayoría de edad), catálogo y carrito, pago Mercado Pago, **pick-up con geolocalización** (comparte ubicación en viaje; al bajar el ETA del umbral dispara la preparación), self-checkout en local (escaneo + pago + QR de salida), notificaciones push, puntos.
