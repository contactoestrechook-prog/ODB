# O.D.B Premium Market — Identidad visual

Fuente: `elementos-graficos.ai` (kit de marca provisto por el cliente, junio 2026).

| Color | Hex | Uso |
|---|---|---|
| Rojo O.D.B | `#B82D25` | Color primario: botones principales, header, acentos, badges de oferta |
| Rojo oscuro | `#932A1F` | Hover/presionado del primario, fondos de énfasis |
| Negro | `#000000` | Logo, textos, navegación inferior, botones secundarios |
| Blanco | `#FFFFFF` | Fondo base de la app |
| Crema | `#F0EBE2` | Superficies suaves: cards, fondos de sección, botones terciarios |

Elementos del kit: logo en negro sobre blanco y blanco sobre placa roja ("O.D.B / PREMIUM / Market"), botones pill (borde rojo, relleno rojo, crema y negro), pins de mapa en crema/rojo/negro (usar para el pick-up).

Regla práctica: el rojo es para acción y oferta, el negro para estructura y premium, el crema para descanso visual. Texto sobre rojo o negro siempre blanco; sobre crema siempre negro.
